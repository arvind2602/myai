## 🔌 Installation

-   Clone this repo

```bash
git clone https://github.com/A606A/dentics-landing.git
```

-   Go to the project directory

```bash
cd dentics-landing
```

-   Install dependencies

```bash
npm i
```

## 💻 Usage

Project directory

-   start the server

```bash
npm start
```

-   build the project

```bash
npm run build
```
